Requirements for the toolkit. All project tasks run through Stata (with shell scripts). Some estimators call R from Stata, so both Stata and R should be installed. Start with code/i_master.do after unpacking the zip.
  • Stata packages used include reghdfe, bacondecomp, eventstudyweights, eventdd, and rsource.
  • R packages for Callaway & Sant’Anna / Sun & Abraham are listed in the master do-file and README.
  • Toolkit last updated March 2021.
